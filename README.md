# FreightFlow LLC — Minimal Website

## Deploy on GitHub Pages
1. Create a repo (example: `freightflow-minimal`)
2. Upload:
   - `index.html`
   - `css/styles.css`
   - `js/main.js`
3. GitHub → Settings → Pages → Deploy from branch → `main` / `(root)`

## Quote form (no backend)
The form uses a `mailto:` draft. Add your business email in `js/main.js`:
```js
const to = "dispatch@yourdomain.com";
```
